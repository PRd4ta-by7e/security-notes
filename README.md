# Security Notes

A working reference of notes, cheat sheets, and lab writeups built while training in cybersecurity — SOC/blue team analysis, network enumeration, and offensive security fundamentals.

This is a living repo. Content gets added as I work through labs, courses, and hands-on practice.

> ## ⚠️ Disclaimer — Educational Purposes Only
>
> Everything in this repository is for **educational purposes only**, documented from authorized lab environments (personal labs, licensed training platforms, and sanctioned coursework). None of it is intended for use against any system, network, or account you do not own or have **explicit written authorization** to test.
>
> Unauthorized access to computer systems is illegal under laws including the U.S. Computer Fraud and Abuse Act (CFAA) and equivalent legislation in other countries. The author assumes no liability for misuse of any information in this repository. If you're not sure whether you're authorized to test a system — you're not.

## Contents

- [`tools/`](./tools) — practical reference guides per tool: network scanning (`nmap`), SIEM/SOC workflow (`splunk`), IDS/IPS (`suricata`), and AD/network penetration testing (`responder`, `crackmapexec`, `impacket`, `bloodhound-mimikatz`, `hashcat`, `hydra`, `web-enum`, `metasploit`)
- More tools and lab writeups get added as I work through them

## Why this exists

Most of my study lives in private lab environments and personal notes. This repo is where I pull out the reusable, general-purpose reference material — the stuff worth sharing and worth having version-controlled.
